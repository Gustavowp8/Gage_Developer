Planos para sites

Site simples

O site simples apenas mostra informações para o usuário como imagens e texto, este tipo de site é ideal para trabalhadores autônomos que queiram mostrar o seu trabalho ou para empresas como site institucional.

Saiba mais

Site basico

O site básico além de mostrar informações para o usuário, conta com sistema de mensagens, onde é possível receber mensagem ou e-mail dos clientes, este plano é ideal para quem trabalha autônomo e deseja receber pedidos dos clientes diretamente do site.

Saiba mais

Site Completo

O site completo além de fazer tudo que os sites simples e básico fazem ele também permite publicações como capa e textos dentro do site através de um painel de controle, este site é ideal para quem trabalha com eventos ou é influencer.

Saiba mais

Site FLEX

O site FLEX é para clientes que precisam de um site feito sob encomenda, caso você precise de um site de vendas ou educacional etc, você pode falar com a nossa equipe que nós vamos montar um site conforme você precisa.
